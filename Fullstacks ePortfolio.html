1. What is Full Stack Development?

Full stack development means working on both the front-end and back-end of a web application.

Front-end: HTML, CSS, JavaScript (user interface)

Back-end: Server, APIs, database handling

Example: A login page where user enters details and server verifies them.

2. HTML

HTML is used to structure web content.

<h1>Hello </h1>
<p>This is a paragraph</p>
<a href="https://example.com">Visit Site</a>

HTML provides the basic structure of a webpage.

3. CSS

CSS is used to style HTML elements and improve design.

body {
    background-color: lightblue;
    font-family: Arial;
}
h1 {
    color: darkblue;
}

CSS improves layout, colors, and readability.
